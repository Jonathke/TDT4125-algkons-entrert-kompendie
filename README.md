# TDT4125-algkons-entrert-kompendie
Notater i TDT4125 algoritmekonstruksjon fra våren 2020


Dette kompendiet er sammensatt av notater fra TDT4125 våren 2020.

Målet var å få en "high-level" oversikt over faget, som også gir en enkel inngang til de viktige detaljene.

Hovedsaklig ment for meg selv, men om det hjelper noen andre og er jo det bare bra.

Feel free til å påpeke både faglige- og skriveleif.
